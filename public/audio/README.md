# Musique d'ambiance

Dépose un fichier `ambient.mp3` dans ce dossier pour activer la musique de fond (bouton son dans le HUD).

Je n'ai pas pu en télécharger un automatiquement dans cette session : freepd.com (que j'allais utiliser) a fermé, et je ne voulais pas deviner d'URL de téléchargement au hasard. Le lecteur audio est déjà branché et fonctionnera dès que ce fichier existera — aucun changement de code nécessaire.

Sources gratuites (CC0 / libres de droits) à essayer :
- https://opengameart.org (filtrer par licence CC0)
- https://pixabay.com/music/ (licence Pixabay, pas d'attribution requise)
