# Portfolio-YSF

Portfolio 3D interactif de Youssef Yaslane — un monde miniature en Three.js où le visiteur conduit une voiture à travers différentes zones (Accueil, À propos, Expériences, Projets, Compétences, Formations, Contact) pour découvrir mon parcours.

## Stack

React · Vite · TypeScript · React Three Fiber · Drei · Three.js · GSAP · Tailwind CSS · Framer Motion

## Démarrage

```bash
npm install
npm run dev
```

## Statut

🚧 En cours de construction — le contenu (CV, projets, compétences) sera intégré via un fichier de données JSON central.
